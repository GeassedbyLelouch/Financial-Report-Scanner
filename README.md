# Financial-Report-Scanner
A tool that examines the public financial reports of companies and cross references their officially reported numbers against built in metrics such as Debt to Equity Ratio
## Features
- Fetches public company financial data
- Extracts key financial values
- Calculates common financial ratios
- Displays results in a readable format

## Example Output
Company: MidgetCorp
Calculated Debt-to-Equity: 2.1

## Tech Stack
- Python
- NumPy (planned)
